# 腾讯云翻译API配置说明

## 所需参数

在 `.env.local` 文件中需要配置以下参数：

### 必填参数

1. **TENCENT_SECRET_ID** - 腾讯云SecretId
   - 获取方式：登录腾讯云控制台 -> 访问管理 -> API密钥管理
   - 地址：https://console.cloud.tencent.com/cam/capi

2. **TENCENT_SECRET_KEY** - 腾讯云SecretKey
   - 获取方式：同上，与SecretId一起获取

### 可选参数

3. **TENCENT_REGION** - 腾讯云地域
   - 默认值：`ap-beijing`（北京）
   - 可选值：
     - `ap-beijing` - 北京
     - `ap-shanghai` - 上海
     - `ap-guangzhou` - 广州
     - 其他地域请参考腾讯云文档

4. **TRANSLATION_API** - 翻译API选择
   - 默认值：`baidu`
   - 要使用腾讯云翻译，请设置为：`tencent`
   - 可选值：`tencent`、`baidu`、`google`、`mymemory`、`auto`

## 配置示例

在项目根目录创建 `.env.local` 文件，内容如下：

```bash
# 腾讯云翻译API配置
TENCENT_SECRET_ID=your_secret_id_here
TENCENT_SECRET_KEY=your_secret_key_here
TENCENT_REGION=ap-beijing
TRANSLATION_API=tencent
```

## 使用方法

1. 在腾讯云控制台获取 SecretId 和 SecretKey
2. 创建 `.env.local` 文件并填入上述参数
3. 设置 `TRANSLATION_API=tencent` 以使用腾讯云翻译
4. 重启服务器

## 注意事项

- `.env.local` 文件不会被提交到Git仓库（已在.gitignore中）
- 请妥善保管 SecretKey，不要泄露
- 腾讯云翻译API需要开通机器翻译服务
- 请确保账户有足够的余额或免费额度
